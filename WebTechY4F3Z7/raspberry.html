Raspberry PI 400

Átlalános tudnivalók

A Raspberry PI 400 egy olyan új modell melynek keretein belül, ha asztali számítógépet szeretne használni, nem szükséges mindenféle körülményes megoldásokat alkalmazni.

Az eszköz eleve úgy van kialakítva hogy a csomagból való kibontás után egy könnyed összeszerelés követően egyből használható is. Így már nincs szüksgé házilag barkácsolt case-ek gyártására se egyébb nyűglődésre.

Az eszköz rendelkezik a megfelelő erőforrásokkal és szoftverrel ahhoz hogy tényleges aztali számítógépként lehessen használni. Mind emellett rendelkezik a lényeges hardveres funkciókkal mint másmodellek (pl:GPIO)

Mint a fent látható animáció is mutatja a rendelést követően házhoz szállítjuk a termékünket így önnekki sem kell mozdulni ahhozhogy hozzáférjen eme remek géphez!

Rendelje meg a készletet az alábbi linken: https://malnapc.hu/yis/raspberry-pi/pi-400

Specifikációk és egyéb modellek

  • CPU: megnövelt órajelű, 1,8GHz-es quad.core ARM Cortex-A72
  • Gyári passzív CPU hűtési technológia
  • RAM: 4GB LPDDR4-3200 DRAM
  • GPU: VideoCore VI graphics (OpenGL ES 3.1, Vulkan), 4kp60 HEVC decode
  • LAN: True Gigabit Ethernet
  • Wireless LAN: 2,4GHz és 5GHz-es kétsávos IEEE 802.11 b/g/n/ac wifi
  • Bluetooth: 5.0 BLE
  • USB: 2× USB 3.0 és 1× USB 2.0 portok
  • HDMI: 2× micro-HDMI port (1× 4kp60 vagy 2× 4kp30)
  • Tápellátás: USB-C 5V 3A-es gyári tápegységről
  • GPIO: Raspberry Pi kompatibilis 40-pines horizontális GPIO csatlakozó
  • Raspberry PI 3-as család
    • Raspberry PI 3 A+
    • Raspberry PI 3 B+
  • Raspberry PI 4-3s család
    • Raspberry PI 4 B 2GB
    • Raspberry PI 4 B 4GB
    • Raspberry PI 4 B 8GB
